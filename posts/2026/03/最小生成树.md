# 最小生成树 MST

## 定义

- 无向带权图

- n 个节点 n-1 条边（连通且无环）

- 边权和最小

## Kruskal (加边)

- 将边按边权排序

- 从小到大选边，若一条边所连接的两点已经连通（加上此边会成环）则跳过

- 选 n-1 条边

- 并查集判断连通

时间复杂度：*O(MlogM)* 其中 M 为边的数量

瓶颈在于对边排序

```cpp
// 默认 n 个点的编号从 0 ~ n-1

// 并查集
class UF{
    vector<int>fa;
    int cnt;//连通分量数量
public:
    UF(int n):fa(n),cnt(n){
        for(int i=0;i<n;++i){
            fa[i]=i;
        }
    }
    int find(int x){
        return fa[x]==x?x:fa[x]=find(fa[x]);
    }
    bool merge(int a,int b){
        a=find(a);
        b=find(b);
        if(a==b)return 0;
        fa[a]=b;
        cnt--;
        return 1;
    }
    int count(){
        return cnt;
    }
};
// 边
struct edge{
    int u,v;
    int w;
};
// 连通：返回最小生成树的边权和
// 不连通：返回 -1
int kruskal(int n,vector<edge>&Edges){
    UF st(n);
    int sum=0;
    for(auto&[u,v,w]:Edges){
        if(st.merge(u,v)){
            sum+=w;
        }
    }
    if(st.count()==1){
        return sum;
    }
    return -1;
}
```

## Prim（加点）

- 选一个起点作为最初的 “生成树”

- 将距离该 “生成树” 最近的点合并进来

- 选 n 个点

- 优先队列维护点到生成树的距离

时间复杂度：*O(MlogM)* 其中 M 为边的数量

瓶颈在于维护优先队列

```cpp
// 默认 n 个点的编号从 0 ~ n-1

// 链式前向星建图
const int N=5005;
const int M=400005;

int head[N];
int nex[M];
int to[M];
int weight[M];
int tot;

void init(int n){
    tot=0;
    for(int i=0;i<n;++i){
        head[i]=-1;
    }
}
void addEdge(int u,int v,int w){
    nex[tot]=head[u];
    to[tot]=v;
    weight[tot]=w;
    head[u]=tot++;
}
// 连通：返回最小生成树的边权和
// 不连通：返回 -1
int prim(int start,int n){
    // 优先队列存 { 点到 “生成树” 的距离，点的编号 }
    priority_queue<pair<int,int>,vector<pair<int,int>>,greater<>>pq;
    // vis[i]=1 : 编号为 i 的点在 “生成树” 中
    vector<char>vis(n,0);
    int sum=0;
    int cnt=0;
    pq.push({0,start});
    while(pq.size()){
        auto [d,u]=pq.top();
        pq.pop();
        if(vis[u])continue;
        vis[u]=1;
        sum+=d;
        cnt++;
        if(cnt==n){
            return sum;
        }
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            if(!vis[v]){
                pq.push({weight[ei],v});
            }
        }
    }
    return -1;
}
```
