# 最短路

默认 n 个点的编号为 0 ~ n-1

## Floyd

- 基于邻接矩阵

- 得到图中**任意两点间**的最点距离

- 可处理负边，不能处理负环

- 时间复杂度 $O(N^3)$ 

```cpp
const int N=105; // 点的数量

// dis[u][v] = u->v的距离
int dis[N][N];

const int INF=0x3f3f3f3f;

// 初始化
void init(int n){
    for(int i=0;i<n;++i){
        for(int j=0;j<n;++j){
            dis[i][j]=INF;
        }
    }
}
void floyd(int n){
    for(int k=0;k<n;++k){
        for(int i=0;i<n;++i){
            for(int j=0;j<n;++j){
                if(dis[i][k]!=INF&&dis[k][j]!=INF&&dis[i][j]>dis[i][k]+dis[k][j]){
                    dis[i][j]=dis[i][k]+dis[k][j];
                }
            }
        }
    }
}
```

## 链式前向星建图

空间复杂度：$O(N+M)$

```cpp
const int N=100005; // 点的数量
const int M=400005; // 边的数量
int tot;
int head[N];
int nex[M];
int to[M];
int weight[M];
// 初始化
void init(int n){
    tot=0;
    for(int i=0;i<n;++i){
        head[i]=-1;
    }
}
// 加有向边 u -> v,边权为 w
void addEdge(int u,int v,int w){
    nex[tot]=head[u];
    to[tot]=v;
    weight[tot]=w;
    head[u]=tot++;
}
// 遍历从 u 出发的边
for(int ei=head[u];ei!=-1;ei=nex[ei]){
    int v=to[ei];
    int w=weight[ei];
}
```

## Dijkstra

- 给定起点，得到起点到每个点的最短距离

- 不能处理负边

时间复杂度 $O(MlogM)$

```cpp
// dis[v] = 起点 -> v 的最短距离
int dis[N]; 

// 堆里存 { cost, u } :起点 -> u 的距离为 cost
priority_queue<pair<int,int>,vector<pair<int,int>,greater<>>>pq;

// vis[u]=1 : 起点到 u 在最短距离已结算
bool vis[N];


// 初始化
void init(int n){
    for(int i=0;i<n;++i){
        dis[i]=INF;
        vis[i]=0;
    }
}
void dijkstra(int start){
    dis[start]=0;
    pq.push({0,start});
    while(pq.size()){
        auto [c,u]=pq.top();
        pq.pop();
        if(vis[u])continue;
        vis[u]=1;
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            int w=weight[ei];
            int nc=c+w;
            if(dis[v]>nc){
                dis[v]=nc;
                pq.push({nc,v});
            }
        }
    }
}
```

## SPFA

- 给定起点，得到起点到每个点的最短距离

- 可以处理负边，可以检查负环

时间复杂度 $O(NM)$ ，但在随机图上常数较好

```cpp
// dis[v] = 起点 -> v 的最短距离
int dis[N]; 

// 队列存可能松弛的点
queue<int>q;

// in[u]=1 ：点 u 在队列中
bool in[N]; 
 
void init(int n){
    for(int i=0;i<n;++i){
        in[i]=0;
        dis[i]=INF;
    }
}
void spfa(int start){
    dis[start]=0;
    q.push(start);
    in[start]=1;
    while(q.size()){
        int u=q.front();
        q.pop();
        in[u]=0;
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            int w=weight[ei];
            if(dis[v]>dis[u]+w){
                dis[v]=dis[u]+w;
                if(in[v]==0){
                    q.push(v);
                    in[v]=1;
                }
            }
        }
    }
}
```

**检查负环**

一共有 $n$ 个节点，若无负环，每个节点最多松弛 $n-1$ 次

记录每个节点松弛了多少次

若发现某个节点松弛次数大于 $n-1$ ，说明有负环

若有规定起点 $s$ ，以 $s$ 为起点跑 **SPFA**

若没有规定起点，添加一个 **虚拟起点 S**

**虚拟起点 S** 连接其它所有的节点，边权为 **0**

```cpp
//记录每个节点的松弛次数
int cnt[N];
// 传入起点 s 和节点总数 n，有负环返回 1，否则返回 0
bool spfa(int s,int n){
    q.push(s);
    in[s]=1;
    cnt[s]=1;
    dis[s]=0;
    while(q.size()){
        int u=q.front();
        q.pop();
        in[u]=0;
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            int w=weight[ei];
            if(dis[v]>dis[u]+w){
                if(cnt[v]==n)return 1; // 发现负环
                cnt[v]++;
                dis[v]=dis[u]+w;
                if(in[v]==0){
                    q.push(v);
                    in[v]=1;
                }
            }
        }
    }
    return 0; // 无负环
}
```
